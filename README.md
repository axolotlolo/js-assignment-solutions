# JavaScript Assignment Solutions

Solutions to JavaScript practice questions covering fundamental concepts.

## Questions Covered

1. **question1.js** - Variables and Output
2. **question2.js** - Functions: Calculator
3. **question3.js** - Conditional Statements
4. **question4.js** - Loops and Iteration
5. **question5.js** - Arrays
6. **question6.js** - Objects
7. **question7.js** - To-Do List (CRUD Operations)
8. **question8.js** - JSON Operations
9. **question9.js** - Asynchronous Programming
10. **question10.js** - Async/Await and Promises
11. **math.js & app.js** - Bonus: Modules

## Running the Solutions

Each file can be run independently using Node.js:

```bash
node question1.js
node question2.js
# ... and so on
```

For the module files (math.js and app.js), use:

```bash
node app.js
```

Note: Ensure you're using Node.js version that supports ES modules, or add `"type": "module"` to package.json.
